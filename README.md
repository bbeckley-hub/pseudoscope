## ūüéĮ **Overview**

**PseudoScope** is an automated, locally‚ÄĎexecutable computational pipeline designed specifically for comprehensive *Pseudomonas aeruginosa* genomic surveillance. It integrates **seven essential analysis modules** into a single, cohesive workflow:

- **FASTA QC** ‚Äď Assembly quality metrics (N50, GC%, contig stats)
- **MLST (Oxford scheme)** ‚Äď Multi‚ÄĎlocus sequence typing via 7 housekeeping genes
- **PAST serotyping** ‚Äď O‚ÄĎantigen typing (pasty / camlhmp‚ÄĎblast‚ÄĎregions)
- **AMRfinderPlus** ‚Äď Comprehensive antimicrobial resistance gene detection
- **ABRicate** ‚Äď Multi‚ÄĎdatabase screening (resistance, virulence, plasmids, biocides)
- **Ultimate Reporter** ‚Äď Gene‚ÄĎcentric integration with interactive HTML
- **Visualisation Dashboard** ‚Äď Publication‚ÄĎready interactive plots (PCA, networks, boxplots)

PseudoScope runs entirely **locally** (or on HPC clusters), protects your data privacy, and produces beautiful interactive reports in **minutes**.

---

## ‚ú® **Key Features**

### ūüĒ¨ **P. aeruginosa‚ÄĎSpecific Innovations**

- **Automated critical gene flagging** ‚Äď Carbapenemases (KPC, NDM, VIM, IMP, OXA), colistin (mcr‚ÄĎ1 to mcr‚ÄĎ10), ESBLs (PER, VEB, BEL), 16S rRNA methyltransferases (armA, rmt)
- **MLST (Oxford scheme)** ‚Äď Uses `acsA, aroE, guaA, mutL, nuoD, ppsA, trpE`
- **PAST serotyping** ‚Äď Up‚ÄĎto‚ÄĎdate O‚ÄĎantigen reference database (Robert Petit / camlhmp)
- **Cross‚ÄĎgenome pattern discovery** ‚Äď ST‚ÄĎO combination table, gene co‚ÄĎoccurrence, PCA
- **Gene‚ÄĎcentric views** ‚Äď Each gene table shows **all genomes** that carry it (no truncation)

### ūüöÄ **Performance Advantages**

- **Complete analysis** in **10‚ÄĎ15 minutes** for 5 genomes (16 cores)
- **Linear scaling** ‚Äď 100 genomes in ~2 hours
- **Low memory footprint** ‚Äď Runs on 4GB RAM, scales to HPC
- **Parallel execution** ‚Äď QC, MLST, and PAST run concurrently; ABRicate and AMR sequentially

### Singularity / Apptainer (for HPC clusters, no sudo)

**Option A ‚Äď Build directly from Docker Hub:**

```bash
singularity pull pseudoscope.sif docker://bbeckleyhub/pseudoscope:latest
```

**Option B ‚Äď Convert a local Docker image (if network issues on HPC):**

On a machine with Docker:
```bash
docker pull bbeckleyhub/pseudoscope:latest
docker save bbeckleyhub/pseudoscope:latest -o pseudoscope.tar
singularity build pseudoscope.sif docker-archive://pseudoscope.tar
scp pseudoscope.sif user@hpc.cluster:~
```

**Run on HPC (crucial: `--writable-tmpfs` allows temporary writes inside the container):**

```bash
singularity run --writable-tmpfs -B $(pwd):/data pseudoscope.sif \
  -i "/data/*.fna" -o /data/output --threads 8
```

All output files will be owned by **your HPC user** ‚Äď no `sudo chown` needed.

## ūüöÄ **Usage Guide**

### Basic Commands

```bash
# Single genome
pseudoscope -i genome.fna -o results/

# Batch processing with glob pattern
pseudoscope -i "*.fna" -o batch_results --threads 16

# Skip specific modules
pseudoscope -i "*.fasta" -o results --skip-qc --skip-viz

# Update AMR database only
pseudoscope --update-amr-db

# Show version
pseudoscope --version
```

### All Command‚ÄĎline Flags

| Flag | Description |
|------|-------------|
| `-i`, `--input` | Input FASTA file(s) ‚Äď supports glob patterns (e.g., `"*.fna"`) |
| `-o`, `--output` | Output directory for all results |
| `-t`, `--threads` | Number of CPU threads (default: 2) |
| `--quiet` | Suppress all non‚ÄĎerror output |
| `--verbose` | Show full command output from modules |
| `--version` | Show version and exit |
| `--update-amr-db` | Update AMRfinderPlus database and exit |
| `--skip-qc` | Skip FASTA QC |
| `--skip-mlst` | Skip MLST |
| `--skip-past` | Skip PAST serotyping |
| `--skip-abricate` | Skip ABRicate |
| `--skip-amr` | Skip AMRfinderPlus |
| `--skip-summary` | Skip ultimate reporter |
| `--skip-viz` | Skip visualisation dashboard |

**Supported FASTA formats:** `.fna`, `.fasta`, `.fa`, `.fn`

## ūüďą **Ultimate Reporter & Visualisation**

### **Ultimate Reporter HTML** ‚Äď the heart of PseudoScope

- **Gene‚ÄĎcentric tables** ‚Äď Every AMR or virulence gene is shown with the **complete list of genomes** that carry it (no truncation).
- **Sortable & searchable** ‚Äď Click column headers to sort; type in the search box to filter.
- **Filter buttons** ‚Äď One‚ÄĎclick filters for:
  - Carbapenemases (KPC, NDM, VIM, IMP, OXA‚ÄĎ48‚ÄĎlike)
  - Colistin resistance (mcr‚ÄĎ1 ‚Ä¶ mcr‚ÄĎ10)
  - ESBLs (GES, PER, VEB)
  - 16S rRNA methyltransferases (armA, rmt)
  - Efflux pumps, aminoglycoside modifying enzymes, etc.
- **ST‚ÄĎO combination table** ‚Äď Shows each ST and O‚ÄĎtype pair with the list of samples.
- **FASTA QC integration** ‚Äď N50, GC%, total length displayed alongside typing results.
- **CSV export** ‚Äď Every table can be exported with one click.
- **AI‚ÄĎassisted guide** ‚Äď Suggests questions for ChatGPT / Claude / Gemini.

### **Visualisation Dashboard** ‚Äď interactive plots for publication

- **ST and O‚ÄĎtype bar charts** ‚Äď Counts per type.
- **Top 20‚ÄĎ50 AMR & virulence genes** ‚Äď Stacked bar charts by database (adjustable).
- **Gene co‚ÄĎoccurrence network** ‚Äď Top 100 edges, interactive (hover, drag, zoom).
- **FASTA QC box plots** ‚Äď GC%, AT%, N50, total length with all points (jitter) and outliers.
- **PCA of gene presence/absence** ‚Äď Colored by ST, explained variance labels.
- **Co‚ÄĎoccurrence heatmap** ‚Äď Pairwise counts of top 30 AMR genes.

All plots are generated with **Plotly** ‚Äď fully interactive, downloadable as PNG/SVG, and embedded in a single scrolling HTML file.

## ‚Ěď **Not Happy? Try These Alternatives**

PseudoScope is optimised for *P. aeruginosa* and integrates many databases out of the box. However, if you need a different workflow, consider these excellent tools:

- **[Bactopia](https://github.com/bactopia/bactopia)** ‚Äď A flexible, multi‚ÄĎspecies pipeline for bacterial genomes (supports raw reads, uses Nextflow).
- **[Nullarbor](https://github.com/tseemann/nullarbor)** ‚Äď A read‚ÄĎto‚ÄĎreport pipeline for public health microbiology (great for outbreak surveillance).
- **[Pathogenwatch](https://pathogen.watch)** ‚Äď A web‚ÄĎbased platform for genomics surveillance (requires upload, but very user‚ÄĎfriendly).

Each has its own strengths; PseudoScope focuses on **speed, local execution, and P. aeruginosa‚ÄĎspecific features**.
